This set of controls enables the robot to be selected, its speed to be set, and excursions to be performed.

Select Robot

This dropdown enables a robot to be selected.

Set Speeds

This dropdown allows the rotational and driving speeds to be set.

Route

This button will initiate a manouvre around the entire route.
If a previous excursion did not complete, then a prompt will appear with options to continue or restart.

Skip

This button will skip a node in the route and send the robot to the next destination.

Drive

This button will initiate a driving operation to any location set manually.

Cancel

This button will cancel the current excursion.

Pause

This button will pause the excursion to enable terms and rules to be scrutinised.

Step

This button becomes active when the excursion is paused. It steps the motion on to the next step, allowing detailed debugging of the strategy.

Reset

The reset button performs a number of tasks:
  • Cancels driving
  • Resets the position of any virtual mower
  • Clears down the quality statistics
  • Clears the snapshot buffer
  • Resets the Viewport
If the vision system is having difficulty locating the real robot, it may be latched onto a noise contour, and a reset may resolve the situation.

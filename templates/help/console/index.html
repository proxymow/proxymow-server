Robot Console Feature

The Robot Console is a simple form enabling commands to be sent to the robot.

The feature comprises:

  • An input box where commands can be typed
  • A dropdown list of common commands
  • A submit button
  • A pane to display the command response
The Robot Console offers a richer interface compared to the Supervisor's Direct Commands. Any command supported by the robot can be sent, both synchronous and asynchronous. Commands can be repeated more easily, and the response can be verified.
